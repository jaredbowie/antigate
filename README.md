# antigate

A Clojure library for interacting with the antigate api

## Usage

#+BEGIN_SRC clojure
[antigate "0.1.0"]
#+END_SRC

#+BEGIN_SRC clojure
(ns my-app.core
(:require [antigate.core :as antigate]))
#+END_SRC
